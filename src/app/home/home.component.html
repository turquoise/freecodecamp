FreeCodeCamp Front End Development Projects

The following Front End Development Projects were created in Angular 4 as part of the FreeCodeCamp Front End Development Certificate.

Quote

A Quote page constructed using a basic Quote API and a random function.

Tribute

This is a tribute to Johannes Itten, a Colour Theorist. The data has been extracted from the Wikipedia API and relevant books have been extracted using the Google Books API.

Weather

The Local Weather page was constructed using an Open Source API to access IP and longitude and latitude of the local machine.

Twitch

The Twitch page was constructed using the twitch API and a small selection of twitch users. The page categorises users according to whether or not they are online and which game they are playing.

Wikipedia

The Wikipedia page demonstrates how the Wikipedia API can be used to search for a limited number of results.

Calculator

The Calculator is an example of a basic implementation of a calculator.

Pomodoro Clock

The Pomodoro Clock is a basic implementation of a timer enabling the user to set a 25 minute timer, take short and long breaks and stop the timer.

Tic Tac Toe

The Tic Tac Toe is a basic implementation of a Tic Tac Toe game allowing two players. However, it does not yet have an AI implementation.

Simon Says

The Simon Says game emulates a child's toy which flashes lights and you have to click on the lights in the correct order in order to win the game.
